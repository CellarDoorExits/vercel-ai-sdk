# Contributing

Thanks for your interest in contributing to Cellar Door!

## Getting Started

1. Fork the repo
2. Clone your fork
3. Install dependencies: `npm install`
4. Run tests: `npm test`
5. Make your changes
6. Submit a pull request

## Code of Conduct

Be kind. Be constructive. We are all here to build something useful.
